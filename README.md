## ✨ Sobre o projeto

Sistema de gestão para assistência técnica de lavanderias, com foco em:

- organização de atendimentos
- controle de técnicos
- agendamentos
- integração com calendário

## 🎨 Identidade Visual

A identidade da marca foi desenvolvida junto com o sistema, garantindo consistência entre produto e comunicação.
